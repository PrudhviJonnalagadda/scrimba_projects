BLOCK, INLINE, INLINE-BLOCK

Block

- stack on top of each other
- div, p, h1, etc... are all example of block elements
- set height and margin top and bottom

Inline

- sit side by side
- span and a are all examples of inline element
- can't set height and margin top and bottom

Inline-block

- sit side by side
- button and input are examples of inline-block
- set height and margin top and bottom.
